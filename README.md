# 공공주택 통합 관리 시스템

공공주택의 공급, 입주, 단지 운영관리 업무를 하나의 흐름으로 처리하는 통합 관리 시스템.

## 주요 기능

- 공공주택 공고 및 신청 관리
- 입주민 및 세대 관리
- 관리사무소 대시보드 및 업무 관리
- 중앙관리자 대시보드 및 업무 관리

## 기술 스택

### Backend
- Java 21
- Spring Boot
- Spring Security
- MyBatis
- Oracle Database

### Frontend
- JSP
- JavaScript / Ajax
- React
- Chart.js

### Infra / Tools
- GitHub
- SVN
- Jenkins
- Tomcat
- Redmine
- Notion

## 담당 역할
관리사무소 모듈 중심으로 시설자산, 협력업체, 시설계약, 점검이력, 검침이력 관리 기능을 구현.  
또한 화면과 DB 간 정합성 검증, 테스트 데이터 구성, 권한별 접근 제어 흐름 점검.
